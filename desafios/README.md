# ⚔️ Desafios — Módulo 01: Fundamentos da Linguagem

Esta seção contém **10 desafios práticos** referentes ao Módulo 01 do Grimório Node.

O objetivo desses exercícios é consolidar os fundamentos da linguagem JavaScript estudados durante as aulas, aplicando conceitos como:

- Variáveis (`const`, `let`, `var`)
- Tipos de dados
- Arrays
- Objetos
- Estruturas básicas da linguagem

---

## 🎯 Objetivo

Os desafios foram criados para verificar se os conceitos realmente foram compreendidos, incentivando a prática e o raciocínio lógico.

A recomendação é tentar resolver **sem consultar as respostas ou materiais externos**, utilizando apenas o conhecimento adquirido nas aulas.

---

## 📚 Estrutura

Cada desafio foi pensado para aumentar gradualmente o nível de dificuldade:

1. Desafios 1–3 → nível básico  
2. Desafios 4–7 → nível intermediário  
3. Desafios 8–10 → nível avançado

---

## ⚠️ Regras

- Tente resolver todos os desafios antes de pesquisar soluções.
- Foque na clareza do código.
- Utilize comentários para explicar sua lógica quando necessário.

---

## 🧠 Filosofia do Grimório

Assim como um grimório de magia registra feitiços e conhecimentos, este repositório documenta a evolução no aprendizado de **JavaScript e Node.js**, transformando cada exercício resolvido em mais experiência prática.
